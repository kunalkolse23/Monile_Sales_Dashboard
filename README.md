# 📊 Sales Dashboard Analysis (Power BI)

## 🚀 Project Overview

This project is an interactive sales dashboard built using Power BI. It provides insights into sales performance, customer behavior, and transaction patterns.

## 📌 Key Features

* KPI cards showing Total Sales, Quantity, Transactions, and Average
* Interactive month filter (slicer)
* Sales trend analysis over time
* Geographic sales distribution using map visualization
* Top performing mobile models analysis
* Payment method distribution (UPI, Credit Card, Debit Card, Cash)
* Customer rating breakdown

## 🛠️ Tools & Technologies Used

* Power BI
* Data Cleaning & Transformation
* Data Modeling
* Data Visualization

## 📈 Key Insights

* UPI is the most used payment method
* Certain cities contribute significantly to total revenue
* Sales trends vary across months
* A few mobile models dominate sales performance

## 🎯 Objective

To transform raw sales data into meaningful insights and build an interactive dashboard for better decision-making.
